## Welcome to the Redo API

The Redo API provides programmatic access to manage returns, exchanges, refunds, shipments, and invoices for your store.

**Base URL:** `https://api.getredo.com/v2.2`
**Version:** 2.2.1

## Quick Start

```bash
curl -X GET "https://api.getredo.com/v2.2/stores/{storeId}/returns" \
  -H "Authorization: Bearer YOUR_API_SECRET"
```
